01.02 - Formatações no HTML 5

Seguem alguns exemplos de formatação em HTML 5.

Para utilizar negrito podemos utilizar esta tag que o texto ficara em negrito.

Para utilizar o itálico usamos a tag em para o texto ficar em itálico.

Para sublinhado usamos a tag u para sublinhar o texto.

Para marcar uma parte em destaque usamos a tag mark para dar o destaque.

Se quisermos um texto aumentado usamos esta tag para dar o destaque mas lembrando que ela logo cairá em desuso.

Já para um texto menor usamos a tag small em uma parte do texto para diminuir.

Para um texto riscado usamos a tag del que pode indicar um texto deletado.

Para um texto sobrescrito usamos esta tag sup, como por exemplo em fórmulas, X2+2 = 6.

Ao contrairo temos subscrito para textos na parte inferior, como exemplo H2O.

Para uma pagina que exista uma linha de programa, podemos usar uma tag que represente que a linha é uma linguagem de programação.

nome = str(input("Digite seu nome: ")

Ja para uma sequencia de comandos, com os devidos alinhamentos podemos utlizar de outra maneira.

        
            num = int(input("Digite um numero:"))
            if num % 2 = 0
                print("Seu numero é par.")
            else:
                print("Seu numero é impar.")
            print("Fim do Programa")

Para informar ou explicar o significado de uma abreviação utilizamos a tag (abbr), por exemplo: KFC ou CPF

Para criarmos listas dentro do HTML temos algumas formas:

Listas Ordenadas que possuem uma sequência.

  1. Ligar o aparelho
  2. Sintonizar a frequência
  3. Verificar o ruído
  4. Responder os chamados
  5. Encerrar o contato

Podemos criar tambem listas sem sequencia definida:

  • Acoradar
  • Levantar
  • Lavar o rosto
  • Tomar café
    1. Fazer o Cafe
    2. Esquentar o Leite
    3. Esquentar o Pão
  • Escovar os dentes
